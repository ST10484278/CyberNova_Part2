# 📌 CyberNova ChatBot

## 🧠 Project Description
CyberNova is a WPF-based cybersecurity chatbot that helps users learn about online safety topics such as phishing, malware, password security, and safe browsing through interactive conversation, sentiment detection, and personalized responses.

## 🚀 Features Implemented (Part 2)

- Conversational chatbot using keyword-based response system  
- Sentiment detection (Worried, Curious, Frustrated, Happy, Neutral)  
- Personalized responses using stored user name  
- Follow-up conversation handling (“tell me more”, “explain more”)  
- Memory system for storing user interactions  
- Clean layered architecture:
  - ChatBot (core logic)
  - KeywordResponder (knowledge base)
  - SentimentDetector (emotion analysis)
  - MemoryStore (data storage)
- ASCII art startup display  
- Audio greeting using `greeting.wav`  
- Cybersecurity topics (phishing, malware, scams, passwords, privacy, etc.)  
- WPF chat UI with sender/receiver message bubbles  
- Enter key support for sending messages  
- Scrollable chat interface
